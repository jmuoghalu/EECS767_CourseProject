Summary[edit]

This is a map of the insular areas of the United States.

The first version of this map was based off Image:BlankMap-World-v6.png by User:Astrokey44 with the locations of a few islands added; the source for this are two CIA maps; the maps are a map of the Pacific region and a map of the United States Pacific Island Wildlife Refuges. Hawaii has also been added to the map. Names of the islands are from the CIA Factbook. The second version is based off Image:Dependent_territories_2007.png by User:Sesmith. Additional attribution includes User:Hoshie and User:KronosLine. In this current version, the map incorporates all of the above in an SVG re-creation using File:OCDE World Map.svg by User:Mark Miller. This map is color-coded according to each island's status:

   The 50 states and the District of Columbia
   Incorporated, unorganized territory
   Unincorporated territory with Commonwealth status
   Unincorporated, organized territory
   Unincorporated, unorganized territory

For the PNG image this maps is based off, see File:US insular areas.png. For an uncolor-coded map, see Image:US_insular_areas-B.png. For a version with the freely associated states, see Image:US_insular_areas FAS.png.
